Any lattice path from \((0,0)\) to the point \((k,n-k)\) either passes through the point \((k-1,n-k)\) (and takes one last step to the Right) or the point \((k,n-k-1)\) (and takes one last step Upwards). This is a partition of the set of lattice paths to \((k,n-k)\text{.}\) We also know from Checkpoint 2.3.10 that there are \(\displaystyle\binom{n}{k}\) lattice paths to \((k,n-k)\text{.}\)

Furthermore, by Checkpoint 2.3.10 there are \(\displaystyle\binom{n-1}{k-1}\) lattice paths to \((k-1,n-k)\) and there are \(\displaystyle\binom{n-1}{k}\) lattice paths to \((k,n-k-1)\text{.}\)

Hence by Principle 2.1.7, we must have

\begin{equation*} \binom{n}{k} = \binom{n-1}{k-1} + \binom{n-1}{k}\text{,} \end{equation*}

as desired.
