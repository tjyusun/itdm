Since \(f\) is bijective, we know that it has an inverse \(f^{-1}:B \rightarrow A\text{.}\) To show that \(f^{-1}\) is strictly increasing, we need to show that

\begin{equation*} \text{If } y_1 \lt y_2, \text{ then } f^{-1}(y_1) \lt f^{-1}(y_2) \end{equation*}

for all \(y_1,y_2 \in B\text{.}\)
