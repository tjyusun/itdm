Let \(m_1,m_2 \in \mathbb{N}\) and \(\gcd(m_1,m_2) = 1\text{,}\) and consider the system

\begin{equation*} \begin{cases} x \equiv a \Mod{m_1} \\ x \equiv b \Mod{m_2} \end{cases}\text{.} \end{equation*}

Perform the following steps:

  • Find the multiplicative inverse of \(m_2\) modulo \(m_1\text{,}\) call it \(t\text{.}\)
  • Find the multiplicative inverse of \(m_1\) modulo \(m_2\text{,}\) call it \(s\text{.}\)

A solution to the given system is

\begin{equation*} x \equiv atm_2 + bsm_1 \Mod{m_1m_2}\text{,} \end{equation*}

and this is unique modulo \(M = m_1m_2\text{.}\)
