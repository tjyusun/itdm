Let \(f: A \rightarrow B\) be a bijective function and assume it is strictly increasing. This means that for all \(x_1,x_2\) in \(A\text{,}\)

\begin{equation*} x_1 \lt x_2 \Rightarrow f(x_1) \lt f(x_2)\text{.} \end{equation*}

Since \(f\) is bijective, we know that it has an inverse \(f^{-1}:B \rightarrow A\text{.}\) To show that \(f^{-1}\) is strictly increasing, we need to show that

\begin{equation*} \text{If } y_1 \lt y_2, \text{ then } f^{-1}(y_1) \lt f^{-1}(y_2) \end{equation*}

for all \(y_1,y_2 \in B\text{.}\)

We prove this by contradiction. Take arbitrary \(y_1,y_2 \in B\) such that \(y_1 \lt y_2\text{,}\) and assume that \(f^{-1}(y_1) \geq f^{-1}(y_2)\text{.}\)

Note that \(f^{-1}(y_1) \ne f^{-1}(y_2)\) since \(f^{-1}\) has to be injective; so we must have \(f^{-1}(y_1) > f^{-1}(y_2)\text{.}\)

But since \(f\) is strictly increasing, we have

\begin{align*} f^{-1}(y_1) \amp \gt f^{-1}(y_2) \\ \Rightarrow f(f^{-1}(y_1)) \amp \gt f(f^{-1}(y_2))\\ \Rightarrow y_1 \amp \gt y_2 \end{align*}

and this contradicts the assumption that \(y_1 \lt y_2\text{.}\) Thus \(f^{-1}(y_1) \lt f^{-1}(y_2)\) must be true.
