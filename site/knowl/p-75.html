If \(m\) is odd, then there is a \(k \in \mathbb{Z}\) such that \(m = 2k + 1\text{.}\) Then

\begin{align*} m^3 \amp = (2k+1)^3\\ \amp = 8k^3 + 12k^2 + 6k + 1\\ \amp = 2(4k^3 + 6k^2 + 3k) + 1\text{.} \end{align*}

Since \(k \in \mathbb{Z}\text{,}\) then \(4k^3 + 6k^2 + 3k \in \mathbb{Z}\) as well. Therefore \(m^3\) is odd. This completes the proof.
