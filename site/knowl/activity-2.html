Activity A.4.1.

When writing proofs, one needs to be clear, complete, and correct. Comment on the three proofs below of the statement

\begin{equation*} \text{Let } m \in \mathbb{Z}\text{. Then }m\text{ is even if and only if } m^3 \text{ is even}\text{.} \end{equation*}

Which one(s) is/are

  • convincing?
  • missing steps?
  • easiest to read?
  • correct?

Proof 1 (\(\Rightarrow\)) \(m = 2k \Rightarrow m^3 = 8k^3 = 2(4k^3)\text{,}\) so \(m^3\) is even.

(\(\Leftarrow\)) By contradiction. Assume \(m^3\) is even but \(m\) is odd. Then \(m = 2k + 1 \Rightarrow m^3 =(2k+1)^3 = 8k^3 + 12k^2 + 6k + 1\text{,}\) which is odd.

Contradiction, so \(m\) must be even. So \(m\) even \(\Leftrightarrow\) \(m^3\) even.

Proof 2

\begin{equation*} m \text{ even } \Leftrightarrow m = 2k \Leftrightarrow m^3 = 8k^3 \Leftrightarrow m^3 = 2(4k^3) \Leftrightarrow m^3 \text{ even}\text{,} \end{equation*}

hence proven.

Proof 3 We want to prove that

\begin{equation*} m \text{ is even if and only if } m^3 \text{ is even}\text{.} \end{equation*}

(\(\Rightarrow\)) First, we show \(m\) even \(\Rightarrow\) \(m^3\) even. If \(m\) is even, then we can write \(m = 2k\) for some \(k \in \mathbb{Z}\text{.}\) Then,

\begin{align*} m^3 \amp = (2k)^3\\ \amp = 8k^3\\ \amp = 2(4k^3)\text{.} \end{align*}

Since \(k \in \mathbb{Z}\text{,}\) then \(4k^3 \in \mathbb{Z}\) as well, and so \(m^3\) is even.

(\(\Leftarrow\)) Next, we prove that \(m^3\) even \(\Rightarrow\) \(m\) even. To do this we prove the contrapositive: If \(m\) is odd, then \(m^3\) is odd.

If \(m\) is odd, then there is a \(k \in \mathbb{Z}\) such that \(m = 2k + 1\text{.}\) Then

\begin{align*} m^3 \amp = (2k+1)^3\\ \amp = 8k^3 + 12k^2 + 6k + 1\\ \amp = 2(4k^3 + 6k^2 + 3k) + 1\text{.} \end{align*}

Since \(k \in \mathbb{Z}\text{,}\) then \(4k^3 + 6k^2 + 3k \in \mathbb{Z}\) as well. Therefore \(m^3\) is odd. This completes the proof.
